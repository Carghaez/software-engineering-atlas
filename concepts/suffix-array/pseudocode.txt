# Suffix array: sorted order of all suffix start indices of a string
# (compact index into the text; powers substring search, LCP, BWT, etc.)

function build_suffix_array(s):
    n = len(s)
    sa = [0, 1, ..., n-1]                 # every suffix start position
    sort sa by comparing the suffixes s[i..] lexicographically
    return sa
# Naive comparison sort is O(n^2 log n); production builds (prefix doubling,
# DC3/SA-IS) reach O(n log n) or O(n).

# Example: s = "banana"
#   suffixes: 0:banana 1:anana 2:nana 3:ana 4:na 5:a
#   sorted  : a(5) ana(3) anana(1) banana(0) na(4) nana(2)
#   sa      : [5, 3, 1, 0, 4, 2]
